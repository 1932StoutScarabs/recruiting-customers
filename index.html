Marketing Flyer

This flyer showcases ChattyNotes and could be used for marketing our website. It highlights the key features and benefits of our note-taking platform, designed to help users capture and organize their thoughts more effectively.

User Guide

Page 1: This page has the area to upload files to submit to the AI.
.txt, mp3, mp4, .m4a files can be uploaded. Only 1 file can be uploaded at a time.
Page 2: Page 2 has the AI prompts or personal prompts. This page will send the prompt to the AI.
You can choose from the following prompts: summarize, give me action items, give me notes. It also has a area to ask your own prompt.
Slide 3: This page will show the AI's response. You can save the response to a file or copy it to the clipboard.
Saving the response to a file will save it to the files folder. Copying the response to the clipboard will copy it to the clipboard.
